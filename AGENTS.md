# AGENTS.md

This file contains instructions for AI agents working on this codebase.

## Project Overview

OpenCode plugin that trims thinking/reasoning from context sent to LLMs.

## Commands

- `bun test` - Run tests
- `bun run build` - Type check only (no build needed for plugin)

## Code Style

- TypeScript with strict mode
- No comments unless requested
- Follow existing patterns in codebase

## Testing

All changes require test coverage. Run `bun test` before committing.
